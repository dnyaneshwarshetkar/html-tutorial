-Javascript is only Web Programming Language.

HTML - to define the content of web pages
CSS - to specify the layout of web pages
Javascript - to program the behavior of web pages

-Where to write and Use Javascript?
#1. Inside html elements events
#2. Inside script tag
#3. Inside External Javascript File 

- What Javascript Can Do?
#1. Can Change HTML Content
#2. Can Change HTML Attribute Values
#3. Can Change HTML Styles
#4. Can Hide HTML Elements
#5. Can Show HTML Elements

- JavaScript can "display" data in different ways:

#1. Writing into an HTML element, using innerHTML.
#2. Writing into the HTML output using document.write().
#3. Writing into an alert box, using window.alert().
#4. Writing into the browser console, using console.log().
